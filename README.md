# IT Guru Catalog

Каталог товаров с авторизацией. React 19, TypeScript, Vite, Ant Design, Feature-Sliced Design.

## Запуск проекта

```bash
git clone https://github.com/VVMinin/it-guru.git
```
```bash
cd it-guru
```
```bash
npm i
```
```bash
npm run dev
```

## API

- **Каталог товаров:** [dummyjson.com/products](https://dummyjson.com/docs/products)
- **Авторизация:** [dummyjson.com/auth](https://dummyjson.com/docs/users)

Пример логина и пароля: `emilys` / `emilyspass`

## Страница авторизации

- Поле логина принимает только латиницу и цифры. Пробелы и спецзнаки запрещены. Минимум 4 символа, предел — 20 символов
- Пароль чувствителен к регистру, запрещены пробелы, предел — 16 символов
- Чекбокс «Запомнить данные» сохраняет токен в `localStorage`, иначе — в `sessionStorage`

## Страница каталога

- Сортировки по наименованию, вендору, оценке и цене. Сброс сортировки — кнопка слева от кнопки обновления
- Поиск по товарам с задержкой ввода (debounce)
- Кнопка **+** — анимация нажатия без функционала
- Кнопка **...** — анимация нажатия без функционала

## Модальное окно «Добавить товар»

- **Наименование** — до 50 символов
- **Цена** — до 6 цифр
- **Вендор** — до 20 символов, необязательное. При пустом значении подставляется «—»
- **Артикул** — до 15 символов
