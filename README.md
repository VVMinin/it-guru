Запуск проекта:
git clone https://github.com/VVMinin/it-guru.git
npm i
npm run dev

Запуск в консоли npm run dev.
API каталога https://dummyjson.com/docs/products

API пользователей для авторизации https://dummyjson.com/users
Пример логопаса: emilys/emilyspass

Страница авторизации:
- Поле логина принимает только латиницу и цифры. Пробелы и спецзнаки запрещены. Минимум 4 символа, предел для ввода логина — 20 символов
- Пароль чувствителен к регистру, запрещены пробелы, предел для пароля 16 символов
- Чекбокс «Запомнить данные» сохраняет токен в localStorage, иначе — в sessionStorage

Страница каталога:
- Присутствуют сортировки по наименованию, вендору, оценке и цене, сортировки сбрасываются по кнопке сброса слева от кнопки обновления каталога
- Поиск по товарам с задержкой ввода
- При нажатии на кнопку + происходит анимация нажатия без функционала
- Кнопка ... выводит в консоль "У кнопки нет функционала"

Модальное окно:
- Поле "Наименование" принимает до 50 символов
- Поле "Цена" не принимает больше 6 символов
- Поле "Вендор" не принимает больше 20 символов и является необязательным, при пустом значении по умолчанию будет стоять —
- Артикул принимает не больше 15 символов
