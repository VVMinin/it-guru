Запуск в консоли npm run dev.
API каталога https://dummyjson.com/docs/products

API пользователей для авторизации https://dummyjson.com/users
Пример логопаса: emilys/emilyspass

Страница авторизации:
-Нормализовано поле логина, регистр не играет роли. Пробелы и спецзнаки запрещены. Предел для ввода логина - 20 символов
-Пароль чувствителен к регистру, запрещены пробелы и спецзнаки, предел для пароля 16 символов.

Страница каталога:
-Присутствуют фильтры вендора, оценки и цены, фильтры сбрасываются по кнопке сброса слева от кнопки обновления каталога
-При нажатии на кнопки + и - происходит анимация нажатия без функционала. 

Модальное окно:
- Поле "Наименование" принимает до 50 символов
- Поле "Цена" не принимает больше 6 символов
- Поле "Вендор" не принимает больше 20 символом и является необязательным, при пустом значении по умолчанию будет стоять -
- Артикул принимает не больше 15 символов
